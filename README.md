## Description
Willcrack supports a variety of ciphers
- Caesar (done)
- Affine (done)
- Vigenere (done, maybe add encryption)
- Atbash (done)
- Baconian (done)
- Hill (in progress)
- Porta (in progress)
- Morse (done)
- Binary (done)
- Base64

## Getting Started
- Just download and run
- Open the HTMLs if you would like to have a user interface
