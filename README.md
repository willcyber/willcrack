## Description
Willcrack supports a variety of ciphers
- Caesar (done)
- Affine (done)
- Vigenere (only works if key same length, needa fix)
- Atbash (done)
- Baconian (mostly done, needa fix spaces and incorporate 24 char)
- Hill (in progress)
- Porta (after fix vigenere)
- Morse (done)
- Binary (after I fix baconian)
- Base64
